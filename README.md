* ?meme fetches from /r/doge from this [repo](https://github.com/R3l3ntl3ss/Meme_Api)
* ?doggo fetches random image and blesses you [dog.ceo](https://dog.ceo)
* ?swole places target user's avatar with swole doge head :3
